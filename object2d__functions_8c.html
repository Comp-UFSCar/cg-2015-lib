void drawLine ( struct Point2D *  p1,
struct Point2D *  p2,
struct Window *  win,
struct BufferDevice *  device,
int  color 
)

Draw a straight line given two points.

Two points on space will be first normalized to the window, and then converted to the device space. With the points now set to the device reference, the algorithm tracks and plots a line between the two points.

Parameters
p1A point in space.
p2Another point in space.
winWindow where this should be based on.
deviceBufferDevice where this will be allocated.
colorRGBColor number that will be used based on a Palette.
